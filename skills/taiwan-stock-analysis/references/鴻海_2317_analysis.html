🔍 經營亮點

  • 三年營收 61,622 → 68,596 → 81,031 億，CAGR +14.6%，2025年加速成長 +18.1%，AI伺服器與電動車組裝訂單貢獻顯著
  • 毛利率維持 6.15–6.29%，代工業特性使毛利低但穩定；規模擴張未見結構性毛利改善，吃的是費用攤薄紅利
  • 管銷研費用率從 3.61% 降至 2.95%，研發費用絕對值持續成長（1,100 → 1,217億）但占比下降，反映智慧製造轉型投入
  • 推銷費用極低（273億，占比僅0.34%），符合代工 B2B 商業模式特性，無需大量行銷支出
  • 營業利益三年從 1,665 → 2,006 → 2,592 億，CAGR +24.8%，遠超營收成長速度，費用攤薄效益顯現

🔍 獲利亮點

  • 2025年EPS 13.61元，為近三年最高，YoY +23.6%；淨利 1,894億創新高，配息能力持續強化
  • 淨利率 2.23–2.34%，受代工業毛利結構限制，業外損益（主要為匯兌與投資收益）貢獻約 -0.86個百分點
  • ROE 從 8.25%（2024）回升至 9.61%（2025），淨利大幅成長 +24% 驅動，股東權益成長（8.5%）相對溫和
  • 現金股利支出三年從 587 → 807 → 1,067億，配息積極擴大；以2025淨利1,894億計，配息率約56%
  • 三層利潤率（毛利6.15% → 營業3.20% → 淨利2.34%）中，業外損益為正貢獻（+2.34%-3.20%=-0.86pp差異主因為所得稅與少數股東損益）

🔍 財務健全度亮點

  • 流動比率三年持續下降（159.0% → 155.2% → 146.1%），2025年首度低於150%警戒線，AI伺服器擴產帶動流動負債大增（+27108億），需持續監控短期流動性
  • 負債比率從57.2%上升至61.4%，超過60%警示水位；主因為快速擴張帶動負債增加（22,535 → 31,339億），槓桿程度明顯增加
  • 現金部位10,164億仍屬充裕，較2024年低谷（9,371億）回升，但距2023年高峰（11,977億）仍有差距
  • 2023年營業CF為-982億（異常），係大量積累AI伺服器備料存貨所致；2024年存貨消化後強勁回復至4,456億，現金轉換能力確認
  • 2025年資本支出達1,532億（扣除投資CF計算），為近年最高，反映積極擴建AI伺服器產能；FCF收窄至737億但仍正值，不影響配息能力
